
STM32智能衣柜功能如下：
1.	DHT11检测温湿度、设置阈值进行加热和通风
2.	RTC获取实时时间，并定时进行紫外线UV消毒
3.	通过光电传感器检测柜门状态，关门时消毒，开门时判断光照开灯
4.	光敏传感器采集光强，显示环境数据并根据此判断是否需要开灯
5.	MQ2烟雾传感器检测烟雾浓度是否超标，超标触发蜂鸣器声光报警
6.	按键可调节时间和全部的阈值信息与定时信息
7.	蓝牙APP可远程控制衣柜系统进行执行加热、通风、照明和消毒的功能
8.	蓝牙APP可接收环境的实时信息数据
